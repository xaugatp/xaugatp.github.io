PDF-Based Chatbot with Hugging Face

Developed a chatbot system to process PDF data, create embeddings, and enable intelligent Q&A.

Python, PyPDF2, LangChain, Hugging Face, Solara, ChromaDB

Technologies: Python, PyPDF2, LangChain, Hugging Face, Solara

Vector Database: ChromaDB

Year: 2023

This project involved building an interactive chatbot system capable of answering user queries based on PDF documents. The pipeline included:

  • PDF Processing: Merged multiple PDFs using PyPDF2 and extracted text data.
  • Text Splitting: Split PDF content into manageable chunks using RecursiveCharacterTextSplitter for efficient processing.
  • Vector Database: Created embeddings for document chunks using Hugging Face's `sentence-transformers` and stored them in a persistent ChromaDB.
  • Chatbot Interface: Built a Solara-based UI for real-time question answering using LangChain's RetrievalQA and Hugging Face models.
  • Optimization: Integrated Hugging Face's `Falcon-7B-Instruct` for enhanced conversational responses with reduced latency.
This project showcases expertise in integrating machine learning, vector databases, and user interface design for practical applications.

Local File Organizer

Developed a Python-based tool for organizing files efficiently by content, date, or type.

Python, NLTK, NexaVLM, TensorRT, PyPI

Team Size: 3

Date: 2023

Distribution: Published as a PyPI package

The Local File Organizer is a Python tool designed to categorize and manage files in local directories with various customizable modes:

  • Mode 1: Organize by Content - Utilizes machine learning models to analyze and classify files based on their content.
  • Mode 2: Organize by Date - Groups files by creation or modification dates.
  • Mode 3: Organize by Type - Sorts files into folders based on file types (e.g., images, documents).
My Responsibilities:
  • Implemented the core logic for file classification and organization using Python's os module.
  • Integrated advanced machine learning models (NexaVLM and NexaText) for intelligent content-based file organization.
  • Added support for natural language processing (NLP) with NLTK for analyzing text files.
  • Designed and published the project as a PyPI package for easy installation and use.
  • Ensured robust error handling and created detailed documentation for end-users.
The tool offers both command-line and interactive modes, making it accessible to a wide range of users. Its silent mode allows operation logs to be saved in a text file for future reference.
